# Инструкция по использованию Tag Autocomplete App

## Установка:

1. Установите Python 3.7+ если еще не установлен

2. Установите необходимые библиотеки:
   pip install tkinterdnd2

   Примечание: tkinter обычно идет с Python по умолчанию

## Запуск:

python tag_autocomplete_app.py

## Использование:

1. **Поиск тегов**: 
   - Начните вводить название тега в поле поиска
   - Появятся подсказки с подходящими тегами
   - Используйте стрелку вниз для перехода к списку результатов

2. **Выбор тега**:
   - Двойной клик по тегу в списке
   - Или нажмите Enter когда тег выделен
   - Поле поиска автоматически очистится

3. **Промпт**:
   - Выбранные теги отображаются в нижнем поле
   - Теги разделяются запятыми
   - Нажмите "Копировать в буфер обмена" для копирования

4. **Загрузка CSV файлов**:
   - Кнопка "Загрузить CSV" для выбора файла
   - Или перетащите CSV файл в окно приложения
   - Поддерживается формат: название,тип,количество,алиасы

## Формат CSV:

Файл должен содержать строки в формате:
название_тега,тип,количество_постов,"алиасы_через_запятую"

Пример:
1girl,0,4114588,"1girls,sole_female"
masterpiece,5,1500000,

Где:
- название_тега: основное название
- тип: категория (0-9)
- количество_постов: для сортировки по популярности
- алиасы: альтернативные названия (опционально)

## Горячие клавиши:

- Enter в поле поиска: выбрать первый результат
- Стрелка вниз: перейти к списку результатов
- Enter в списке: выбрать тег

## Файлы:

- tag_autocomplete_app.py - основное приложение
- example_tags.csv - пример CSV файла с тегами
- requirements.txt - зависимости для установки
